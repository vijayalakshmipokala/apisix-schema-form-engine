# APISIX Schema Form Engine - Backend

This is the backend for a Schema Form Engine built with **FastAPI**.  
It generates UI forms dynamically from JSON Schema, supporting:

- Primitive fields (text, number, boolean, checkbox)
- `oneOf` conditional fields
- Field dependencies
